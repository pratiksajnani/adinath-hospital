üë• Manage Users (CRUD)

View All Users

HMS.users.getAll()

Create New User

HMS.users.add({
    email: 'newstaff@adinathhospital.com',
    password: 'staff123',
    name: 'New Staff Member',
    role: 'staff',
    permissions: ['appointments', 'queue'],
    phone: '9876543210',
    active: true
})

Update User

HMS.users.update('U005', {
    name: 'Updated Name',
    phone: '9999888877'
})

Delete User

HMS.users.delete('U005')
‚ö†Ô∏è Be careful! Deleting users is permanent. Back up data first.

HMS.appointments.cancel('A001')
